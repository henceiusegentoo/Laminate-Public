This document describes the internal design and implementation of Laminate, a Paper 1.21.11
fork that adds first-class support for _hybrid plugins_: standard Bukkit/Paper plugins that
may also register SpongePowered Mixin bytecode transformations against the live server
without requiring operators to supply manual JVM flags or install a separate mod loader.

Every question about how the server starts, how mixins are registered, how the plugin
provider system recognises hybrid plugins, and how `laminate-api` tools work for plugin
authors should be answerable from the sections that follow.
